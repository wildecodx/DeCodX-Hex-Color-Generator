DeCodX Hex Color Generator

Hit the spacebar on your keyboard to generate colors.

Usecase Area

  • Click spacebar to randomize 5 different colors.
  • If you want to save just click the color code that starts with # and you got copies instantly.
  • If you have chosen color you can click the lock button and for the next generate color the color you locked remains.
